# Native gates

In this directory we have the code for fetching the set of native gates for AWS Braket devices.

As a result, you must have the AWS Braket account set up on your computer.



### Parameters

You will need to run the `native_gates.ipynb` notebook. There are no parameters for this notebook.

### Usage

As the notebook is set up now, if the required dependencies are installed, you may run the notebook with jupyter notebook by clicking on 'Run All'.

This will print out the list of native gates for AWS Braket devices.
